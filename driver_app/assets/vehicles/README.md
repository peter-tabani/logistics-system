# Vehicle images for the home category cards

Drop your vehicle renders here with these EXACT names and the app uses them
automatically (until then, the cards fall back to simple line icons):

- `bike.png`   — motorbike / bike
- `truck.png`  — truck or van
- `car.png`    — car

## Specs (for the Uber/Bolt look)
- **Format:** PNG with a **transparent background** (so they sit cleanly on the cards)
- **Style:** all 3 the same — same camera angle (3/4 view or top-down), same
  lighting, similar scale. Realistic 3D renders or clean flat illustrations both work,
  as long as the three match.
- **Size:** square, at least **512 x 512 px** (bigger is fine; they'll be scaled down)
- **Content:** just the vehicle, centered, no text/logos/background

That's it — replace these and rebuild. No code changes needed.
