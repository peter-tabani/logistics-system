# Driver profile picture

Drop your photo here with this EXACT name and the app uses it automatically
(until then, the app shows the default avatar):

- `avatar.png`  — the driver's profile picture

## Specs
- **Format:** PNG or JPG (rename to `avatar.png`)
- **Shape:** square (it's cropped to a circle in the app)
- **Size:** at least **256 x 256 px**
- **Content:** face/portrait centered

That's it — drop it in and rebuild. No code changes needed. Used on the home
header and the Profile tab.
